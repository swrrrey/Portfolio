# Personal Portfolio Website 🚀

A responsive, multi-section personal portfolio website built with HTML, CSS, and JavaScript — designed to showcase projects, skills, and contact information for data science and AI roles.


## 📌 About This Project

### Problem
Resumes alone can't fully communicate a developer's skills, personality, or project depth. Recruiters and hiring managers increasingly look for an online presence that shows — not just tells — what a candidate can do.

### Solution
A clean, fully responsive portfolio website that loads fast, works on all screen sizes, and presents projects, technical skills, and contact details in a structured, professional layout.

---

## 🛠️ Tech Stack

| Layer      | Technology             |
|------------|------------------------|
| Structure  | HTML5                  |
| Styling    | CSS3 (Flexbox, Grid)   |
| Behaviour  | Vanilla JavaScript     |
| Deployment | GitHub Pages           |

---

## ✨ Features

- **Responsive navbar** with hamburger menu toggle for mobile devices
- **Hero section** with name, tagline, and CTA button
- **About section** highlighting skills in Python, SQL, Excel, and Power BI
- **Projects section** showcasing:
  - *Forensic Accounting AI* — ML-based financial statement manipulation detector using Python, Pandas, and Power BI
  - *AI & IoT Waste Management System* — Smart waste monitoring using IoT sensors and AI analytics
- **Certifications section** — Data Analytics & Python (Coursera)
- **Contact form** with client-side validation

## 🚀 Getting Started

### View locally
```bash
git clone https://github.com/swrrrey/portfolio-website.git
cd portfolio-website
open index.html     # or double-click the file
```

No dependencies, no build step — opens directly in any browser.
